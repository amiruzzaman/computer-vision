## Overview

This homework is designed to help you understand how **camera-related parameters** affect the output of a computer vision pipeline. You will work with a provided Jupyter notebook, observe its behavior **without any modifications**, and then explore how changing parameters alters the results.

The goal is not only to run code, but to **reason about cause and effect**: *what changed, why it changed, and what you learned from it*.

---

## Learning Objectives

By completing this assignment, you should be able to:

* Correctly run and reproduce results from an existing Jupyter notebook
* Identify key camera or image-processing parameters
* Systematically modify parameters and observe changes in output
* Compare results before and after parameter changes
* Clearly document and explain your observations in a written report
